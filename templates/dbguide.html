DB Guide Database

Welcome to the DB Guide Database created by the Frederick National Laboratory. We have collected data of single guide RNAs (sgRNAs) from CRISPR/Cas9 related publications and have made the data searchable by gene of interest. For more info about the FNLCR, click here.

If you have data that you wish to add to this databases, please download our template file here. Once completed, please e-mail the file to dbGuide@nih.gov. All contributions to this database will be given proper citations.
